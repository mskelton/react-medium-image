# React Medium Image

A medium-style image viewer.

## Installation

```bash
npm install react-medium-image
```

## Usage

```javascript
import MediumImage from "react-medium-image"

const ui = (
  <MediumImage margin={48} zoomIcon={<ArrowsPointingOutIcon />}>
    <img alt="..." src="https://..." />
  </MediumImage>
)
```
